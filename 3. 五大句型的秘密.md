# 533 模型：5 大基本句型

英语句子不是一堆单词的随机堆叠，而是一套有固定「剧本结构」的表达系统。无论句子多长多复杂，剥到最后只剩 5 种内核。本章先用「微型戏剧」的比喻把 5 种句型讲清楚，再为每个句型补充更多例句与拆解分析，帮助你建立稳定的句法直觉。

> 关键认知：一个标准英文句子里，**有且只能有一个核心谓语（动词）**。其余所有成分，要么是配角（宾语、补语），要么是舞台布景（修饰语、从句）。抓住「谁是主角、在做什么、对谁做、变成什么样」，你就抓住了句子的灵魂。

---

## 为什么是「5 种」而不是「无数种」

从语法逻辑看，主语（S）是必须的，谓语（V）是必须的，差异只发生在谓语之后：

- 谓语后面**什么都不接** → 动作不传导（SV）
- 谓语后面接一个**说明主语身份/状态**的成分 → 系表结构（SVC）
- 谓语后面接一个**承受动作的事物** → 直接宾语（SVO）
- 谓语后面接**两个人**：一个接收者 + 一个被传递物 → 双宾语（SVOO）
- 谓语后面接**一个事物 + 它动作后的结果** → 复合宾语（SVOC）

这 5 种组合穷尽了英语「主谓」之后所有可能的信息关系。下面的每一节都给出：配置、逻辑公式、原书例句，以及 3–4 个同类例句与逐步分析。

---

## 模式 1：单人独秀（SV — Subject + Verb）

**配置**：1 个主角，自己完成动作，不涉及任何其他人或物。
**逻辑**：主语 + 不及物动词（intransitive verb，动作不向外传导，后面不接宾语）。

**原书例句**
- `The sun rises.` → 主角「太阳」自己做了「升起」的动作，没有配角。
- `Rain falls.` → 雨自己下，不需要谁接住。
- `Time flies.` → 时间自己流逝，抽象概念也能当主角。

**补充例句与拆解（同类 SV）**

1. `Birds sing.`
   - 拆解：主角 `Birds`（鸟）+ 不及物 `sing`（唱歌）。没有宾语，动作停留在主语自身。
   - 要点：sing 在此是不及物用法，不能加「唱什么」。

2. `The baby cried.`
   - 拆解：主角 `The baby`（婴儿）+ `cried`（哭）。动作内敛，不传递给外部对象。
   - 要点： cried 是不及物，句子完整，无需补出宾语。

3. `My head aches.`
   - 拆解：主角 `My head`（我的头）+ `aches`（痛）。ache 作为不及物，表示主语自身的状态变化。
   - 要点：中文常说「我头痛」，英文把「头」当主语，更符合「主语承受自身状态」的 SV 逻辑。

4. `The clock stopped.`
   - 拆解：主角 `The clock`（钟）+ `stopped`（停了）。stop 作不及物时表示主语自己停下。
   - 要点：stop 也可作及物（stop the car），此处无宾语，判定为 SV。

> 识别口诀：看到动词后**没有名词/代词**紧跟，且动词是不及物或状态类，立刻判定为 SV。

## 模式 2：人设展示（SVC — Subject + Verb + Complement）

**配置**：1 个主角 + 1 个说明主角身份/状态/特征的成分（表语）。
**逻辑**：主语 + 系动词（linking verb：be / become / seem / feel / look / taste 等）+ 表语（名词、形容词或介词短语）。
**本质**：系动词像「等号」，把主语和表语连接成「A = B」的关系，不涉及任何真正的动作。

**原书例句**
- `The sky is blue.` → 天空 = 蓝色标签（系动词 is 相当于等号）。
- `She seems tired.` → 她 = 疲惫状态（seem 是「看起来」的标签）。

**补充例句与拆解（同类 SVC）**

1. `The soup tastes salty.`
   - 拆解：主角 `The soup` + 系动词 `tastes`（尝起来）+ 表语 `salty`（咸的）。
   - 要点：taste 在此是系动词，不是「品尝」的动作；salty 描述主语的属性，不是宾语。

2. `He became a hero.`
   - 拆解：主角 `He` + 系动词 `became`（成为）+ 表语 `a hero`（一名英雄，名词）。
   - 要点：become 表示身份/状态的转变，表语说明转变后的结果。

3. `They look happy.`
   - 拆解：主角 `They` + 系动词 `look`（看起来）+ 表语 `happy`（开心）。
   - 要点：look 作系动词时接形容词，不接副词；若写 `look happily` 则变成「看」这个动作的方式，结构完全不同。

4. `The book is on the desk.`
   - 拆解：主角 `The book` + 系动词 `is` + 表语 `on the desk`（介词短语，表位置）。
   - 要点：表语不只可以是形容词/名词，也可以是介词短语，用来说明主语的位置或状态。

> 识别口诀：动词是 `is/am/are/was/were/become/seem/look/taste/sound` 等系动词时，立刻找后面的「标签」（表语）。

## 模式 3：双人互动（SVO — Subject + Verb + Object）

**配置**：1 个主角 + 1 个承受动作的事物（直接宾语）。
**逻辑**：主语 + 及物动词（transitive verb）+ 直接宾语（动作的承受者）。
**本质**：主角的动作明确地传导到一个对象上。

**原书例句**
- `Students read books.` → 学生（主角）对书（配角）执行「读」的动作。
- `The chef prepares the meal.` → 厨师 → 准备 → 餐点（动作有明确对象）。
- `We visited the museum.` → 我们 → 参观 → 博物馆（空间互动）。

**补充例句与拆解（同类 SVO）**

1. `Lightning struck the tree.`
   - 拆解：主角 `Lightning`（闪电）+ 及物 `struck`（击中）+ 宾语 `the tree`（树）。
   - 要点：struck 把动作精确传导到 tree，tree 是承受者，不是主语。

2. `She wrote a letter.`
   - 拆解：主角 `She` + `wrote`（写）+ 宾语 `a letter`（一封信）。
   - 要点：write 在此为及物，a letter 是写出来的产物，作直接宾语。

3. `The dog bit the mailman.`
   - 拆解：主角 `The dog` + `bit`（咬）+ 宾语 `the mailman`（邮差）。
   - 要点：bite 是及物，动作直接落在 mailman 身上。

4. `Engineers designed the bridge.`
   - 拆解：主角 `Engineers`（工程师）+ `designed`（设计）+ 宾语 `the bridge`（桥）。
   - 要点：design 这类「创作类」及物动词，宾语通常是被创造出来的事物。

> 识别口诀：动词后紧跟一个名词/代词，且能回答「动词 + 什么/谁？」，即为 SVO 的直接宾语。

## 模式 4：送礼模式（SVOO — Subject + Verb + Indirect Object + Direct Object）

**配置**：1 个主角 + 1 个接收人（间接宾语 IO）+ 1 个被传递的东西（直接宾语 DO）。
**逻辑**：主语 + 动词 + 间接宾语（谁收到）+ 直接宾语（收到了什么）。
**本质**：动作涉及「传递路径」——某物从主角转移到另一个人。

**原书例句**
- `Mother gave her son a gift.` → 妈妈 → 给 → 儿子（接收人）→ 礼物（被传递物）。
- `The teacher taught us English.` → 老师 → 教 → 我们（接收人）→ 英语（被传递物）。

**验证测试**：能在间接宾语前加 `to` 或 `for` 且句意不变 → 就是 SVOO。
- `Mother gave a gift to her son.` - `The teacher taught English to us.` 
**补充例句与拆解（同类 SVOO）**

1. `The teacher awarded Jane a medal.`
   - 拆解：主角 `The teacher` + `awarded`（授予）+ IO `Jane`（接收人）+ DO `a medal`（奖牌）。
   - 验证：`awarded a medal to Jane`  → SVOO。
   - 要点：award / give / send / offer / teach / show 这类「给予/传达」动词常带双宾语。

2. `Mom baked us cookies.`
   - 拆解：主角 `Mom` + `baked`（烤）+ IO `us`（我们）+ DO `cookies`（饼干）。
   - 验证：`baked cookies for us`  → SVOO（for 类）。
   - 要点：bake / buy / make / cook 等「为某人做某物」的动词，间接宾语前常加 `for`。

3. `He sent his friend an email.`
   - 拆解：主角 `He` + `sent`（发送）+ IO `his friend` + DO `an email`。
   - 验证：`sent an email to his friend`  → SVOO。
   - 要点：send / mail / text 等「传递信息」动词，天然适合双宾语。

4. `The company offered new employees a bonus.`
   - 拆解：主角 `The company` + `offered`（提供）+ IO `new employees` + DO `a bonus`（奖金）。
   - 验证：`offered a bonus to new employees`  → SVOO。
   - 要点：offer 表「提供」，间接宾语是受益方。

> 识别口诀：动词后接两个名词，且能在第二个名词前加 `to / for` 变成「动词 + 物 + to/for + 人」→ 就是 SVOO。

## 模式 5：改造模式（SVOC — Subject + Verb + Object + Complement）

**配置**：1 个主角 + 1 个承受动作的事物（宾语）+ 1 个描述该事物动作后结果（宾语补语）。
**逻辑**：主语 + 动词 + 宾语 + 宾语补足语（动作让宾语发生了什么变化）。
**本质**：主角不仅作用于宾语，还把宾语「改造成了某种状态或身份」。

**原书例句**
- `They painted the house white.` → 他们刷墙 → 墙变白（动作 + 结果）。
- `We elected him captain.` → 我们选他 → 他当队长。

**验证测试**：能在宾语和补语之间加 `is / are / was`，句意成立 → 就是 SVOC。
- `The house is white.` - `He was captain.` 
**补充例句与拆解（同类 SVOC）**

1. `The election named her president.`
   - 拆解：主角 `The election` + `named`（任命）+ 宾语 `her` + 补语 `president`（总统）。
   - 验证：`she is president`  → SVOC。
   - 要点：name / elect / appoint / make 等「赋予身份」动词，补语说明宾语的新身份。

2. `We painted the wall blue.`
   - 拆解：主角 `We` + `painted` + 宾语 `the wall` + 补语 `blue`（蓝色）。
   - 验证：`the wall is blue`  → SVOC。
   - 要点：颜色类形容词常作结果补语，描述宾语被动作改变后的状态。

3. `The noise drove the dog crazy.`
   - 拆解：主角 `The noise` + `drove`（迫使）+ 宾语 `the dog` + 补语 `crazy`（发疯）。
   - 验证：`the dog is crazy`（在语境下） → SVOC。
   - 要点：drive / push / make 等「使役」动词，补语表示宾语被迫进入的状态。

4. `They considered the plan risky.`
   - 拆解：主角 `They` + `considered`（认为）+ 宾语 `the plan` + 补语 `risky`（有风险的）。
   - 验证：`the plan is risky`  → SVOC。
   - 要点：consider / find / keep / leave 等动词，补语表达「对宾语的判断或保留状态」。

> 识别口诀：动词后接「名词 + 形容词/名词」，且能说「这个名词 is 那个形容词/名词」→ 就是 SVOC。

---

## SVOO 与 SVOC 的一秒区分

| 判断问题 | SVOO（送礼） | SVOC（改造） |
|---|---|---|
| 第三个成分是什么？ | **东西**（礼物、书、钱、信息） | **状态/身份**（白、开心、队长） |
| 验证测试 | 能加 `to/for`：`gave a gift to son` | 能加 `is`：`house is white` |
| 典型动词 | give / send / offer / teach / award | make / keep / paint / elect / name / consider |

> 核心差别：SVOO 的间接宾语是「人」（接收者），SVOC 的宾语补语是「状态/身份」（描述宾语）。一个在传物，一个在改造。

## 五大句型的统一识别流程

读任何英文句，按这四步锁定句型：

1. **找主语（S）**：谁/什么在做这件事？
2. **找核心动词（V）**：发生了什么动作或状态？
3. **看动词后接什么**：
   - 无宾语 → **SV**
   - 系动词 + 标签 → **SVC**
   - 一个名词（动作承受者）→ **SVO**
   - 两个人（接收者 + 物）→ **SVOO**
   - 一个事物 + 它的结果/状态 → **SVOC**
4. **用验证测试确认**：SVOO 加 `to/for`，SVOC 加 `is`。

## 总结

英语语法并非冰冷的规则集合，而是对客观世界逻辑的戏剧化模拟。

**核心公式：一个句子 ≈ 一场戏；一个主语 ≈ 一个主角；一个谓语 ≈ 一个动作。**

通过将语法结构还原为场景互动，学习者可以迅速建立起对英文句法的直觉认知，实现高效的语言习得。下一章我们将看到，长难句不过是在这 5 种基本句型之上「插入」了三类从句（名词性、定语、状语）而已。

# 533 模型：3 个特效 TVM（时态 Tense、语态 Voice、语气 Mood）

动词不是死规则，而是电影的「渲染参数」。本章讲清三个控制维度——**时态（Tense）**管时间轴、**语态（Voice）**管责任链、**语气（Mood）**管真假值。抓准这三个维度，你就能精准控制一个动作「何时发生、谁发起、是事实还是假设」。

> 术语澄清：本章标题里的「语气」对应语言学中的 **Mood（语气）**，不是语音层面的 tone（声调/语调）。英文有陈述、祈使、虚拟三种主要语气；虚拟语气用来表示假设、愿望或正式要求。

---

## 渲染参数总览

| 控制维度 | 管什么 | 核心问题 |
|---|---|---|
| **时态 Tense** | 时间轴 | 什么时候发生？状态如何？ |
| **语态 Voice** | 责任链 | 谁发起的？谁承受的？ |
| **语气 Mood** | 真假值 | 这是事实、命令，还是假设？ |

---

## 时态：手机直播进度条

别背 16 个时态名，抓住两个变量的组合（此分析法受旋元佑《英语魔法师之语法俱乐部》启发）：

- **时间基准（3 种）**：现在 / 过去 / 将来（以说话时刻为原点）；
- **体貌（2 种）**：一般体（把动作当完整点）与完成体（强调截止到某时刻的影响）。

组合得 6 种核心时态：现在一般、现在完成、过去一般、过去完成、将来一般、将来完成。

**用「手机」类比理解**：
- **现在进行时** → 手机正在直播：`I am eating.`（你此刻正在吃，镜头对准你）；
- **过去完成时** → 已存档的录像：`I had eaten.`（饭已吃完，但还能回放）；
- **将来完成时** → 未来存档的预告片：`I will have eaten.`（饭还没吃，系统已存好录像）。

**行动指令**：看到 `am/is/are doing`？问「正在直播吗？」；看到 `had done`？问「已存档的录像吗？」

### 抓住分词的灵魂（-ing 与 -ed）

```
-ing = 主动 + 进行中 / 具备某种属性
    → a running boy（男孩自己在跑）
    → 用于进行时：be + -ing（定时间 + 定主动进行）

-ed = 被动 + 已完成 / 处于某种状态
    → a broken window（窗户被打破）
    → 用于被动语态：be + -ed（定时间 + 定被动完成）
```

## 语态：摄像机位

- **主动语态** = 自己拍自己：`I eat pizza.`（镜头对准我，我吃披萨）；
- **被动语态** = 别人拍你：`Pizza is eaten.`（镜头对准披萨，谁吃？不知道）。

**黄金法则：不知道谁干的？用被动。**
- `The phone was stolen.`（谁偷的？不知道，但手机被偷了）；
- `The report was written.`（谁写的？不重要，报告写完了）。

**验证**：去掉 `by someone`，句子还通顺？→ 就是被动语态。`The report was written [by me]` 去掉 `by me` 依然通顺。

> 语体差异：学术、新闻、公文偏好被动语态，因为它显得客观、去个人化；日常叙事、邮件偏好主动语态，突出「谁干的」。

## 情态动词：给动作加「外挂」

情态动词不是独立规则，而是给动词加上「能力 / 必要 / 建议」等模态：

- **can** → 能力外挂：`I can swim.`（游泳能力已解锁）；
- **must** → 强制外挂：`You must stop.`（停车外挂强制启动）；
- **should** → 建议外挂：`You should study.`（学习外挂可开启）。

**行动指令**：看到 `can/must/should`？立刻问：这是「能开」「必须开」还是「建议开」？

## 语气：事实 vs 故事 vs 假设

**陈述语气（Indicative）**：陈述客观事实。`Water is wet.`（可验证的真值判断）。

**祈使语气（Imperative）**：表达要求、建议、指令。`Hurry up!` `You should rest.`

**虚拟语气（Subjunctive）**：表达与事实相反的假设、愿望或正式建议。核心机制是**「时态倒退（backshift）」**——用语法上的后退来标记「这不是现实」：

| 想表达 | 语法上退到 |
|---|---|
| 与现在相反 | 过去时（*If I were you*） |
| 与过去相反 | 过去完成时（*If I had known*） |

- **与现在相反** → 梦里才成立：`If I were you`（现实中我不是你，但梦里可以说）；
- **与过去相反** → 后悔录像：`If I had known`（当时不知道，现在只能后悔）；
- **建议 / 要求** → 梦里下指令：`He suggested that we start`（现实中没启动，但梦里要启动）。

**终极验证**：看到过去时，但明显不是过去？→ 一定在虚拟语气。`If I were rich`（现实不富，但梦里用 were）；`I wish I were taller`（现实不高，但梦里用 were）。

> 记忆口诀：虚拟语气 = 做梦语法。现实不存在，梦里才用；现在变过去，过去变过去完成。

## 英语本质：电影制作手册

- **五大句型** = 基础镜头（谁在演？演什么？）；
- **时态** = 播放进度条（正在直播？已存档？）；
- **语态** = 摄像机位（自己拍自己？别人拍你？）；
- **语气** = 现实维度（现实？做梦？指令？）。

**行动开关**：下次读句子，立刻问：
1. 这是现实还是做梦？→ 语气；
2. 谁在拍？拍谁？→ 语态；
3. 正在直播？已存档？→ 时态；
4. 谁在演？演什么？→ 句型。

> 别背规则，用「电影思维」拆句子。英语没有例外，只有你还没看懂的「电影参数」。

# 一个公式搞定问题：英文可理解性 = 词汇 × 句型 × 背景信息

读不懂英文，从来不是单一原因。本章先用一个公式把问题结构化，再把「读不懂」拆成三道防火墙（词汇层、句子层、语境层）和 two 个隐藏的思维 Bug，逐一给出破解方法。

---

## 核心公式

> **英文可理解性 = 词汇 × 句型 × 背景信息**

这个公式有扎实的心理学根基。阅读研究中的 **Simple View of Reading（阅读的简单观）** 由 Gough 与 Tunmer 在 1986 年提出，其核心等式是：

```
阅读理解能力 = 解码能力（Decode） × 语言理解能力（Linguistic Comprehension）
```

也就是说，读懂一篇文章 = 能认出词（词汇/解码）× 能理解句子与语篇（句型/语言理解）。二者是**乘法关系**——任一项接近零，整体理解就接近零，如同乘法里缺一个数直接归零。本书把「背景信息」补为第三因子，对应语篇层面的世界知识，使框架更贴近真实阅读场景：

```
英文可理解性 = 词汇（认词） × 句型（拆句） × 背景信息（领域知识）
```

记住这个思维模型，它同样可迁移到其他语言的学习。

## 三道防火墙：英文阅读的「死亡三连」

### 防火墙 1：词汇层——「不认识的路人」

**症状**
- 看到生词直接心态崩了（大脑强制关机）；
- 「假认识」：知道单词，但不知道它在句子里演什么角色；
- 长单词恐惧症：超过 5 个字母就自动糊弄。

**破局补丁：按主题分类背单词**
别按 A–Z 背，按「剧场角色」背：
- **名词 = 演员 / 道具**（"perseverance" = 坚持 → 精神道具）；
- **动词 = CPU**（"replicate" = 复制 → 主角操作道具）；
- **形容词 = 滤镜**（"prevalent" = 普遍的 → 给演员加属性）；
- **连接词 = 剧情转折点**（"because" = 因为 → 开启原因解释）。

**行动指令**：立刻扔掉 ABCD 单词书，改用 **COCA 5000** 高频词表，按「职场」「科技」「学术」等主题背，**覆盖约 95% 的日常语料**（COCA 统计显示，5000 个最高频词约覆盖一般文本的 95%）。

---

### 防火墙 2：句子层——「乱码电影」

**症状**
- 句子超 20 词就头晕（像看没分镜的电影）；
- 单词全懂，连起来像随机乱码；
- 读到句尾忘句头。

**破局补丁：乐高积木拆解法**
英文句子是**树状结构**，不是线性文字。别从左往右死读，先找树干，再拼树枝：
1. **圈动词**（把所有动词标红）；
2. **锁主谓**（没被连接词带的主句）；
3. **括从句**（把从句全塞进括号）；
4. **最后啃修饰词**（树叶）。

**行动指令——拆句三步法：找动词 → 锁主语 → 括号从句**
例：`[In an era] [when digital distractions are prevalent]...` → 先读 `In an era`（在一个时代），再处理 `when...` 这个定语从句。句子层的破解，正是本书 533 模型（5 大句型 + 3 大从句）的主场，详见第三、四章。

---

### 防火墙 3：语境层——「科幻片没看设定」

**症状**
- 单词句型都懂，但读完懵了（「这作者在说啥？」）；
- 被俚语 / 文化梗瞬间卡死；
- 法律文和科技文读起来像两种语言。

**破局补丁：查主题标签法**
读前先问：这是什么「剧场」？

| 文章类型 | 剧场场景 | 你该怎读 |
|---|---|---|
| 法律条文 | 法庭辩论 | 「如果—那么」规则（if–then–else） |
| 新闻报道 | 直播现场 | 最重要信息放最前（倒金字塔） |
| 学术论文 | 实验室报告 | 被动语态 + 客观视角 |
| 小说故事 | 电影放映 | 过去时 + 时间线 |

**行动指令**：读前先扫 3 秒——看标题、开头句、小标题——立刻锁定「剧场类型」，别脑补。

## 两个隐藏 Bug：你的旧代码在拖后腿

### Bug 1：中文语序强迫症

**问题**：你习惯中文「时间 → 地点 → 人物 → 动作」，但英文是「人物 → 动作 → 事物 → 地点 → 时间」。

**补丁：断句重组法**
别翻译成中文，直接按英文原序拼图：
`Because you lied to me yesterday, I'm angry.` → 先读 `you lied`（你骗了我）→ 再读 `I'm angry`（我生气）→ 最后加 `because`（因为）。**别猜，英文所有信息都写在纸上。**

### Bug 2：逐字翻译强迫症

**问题**：非要把每个英文词对应一个中文词，读得像在解密。

**补丁：意群阅读法**
把词块成「信息包」：
- 错：`In / an / era / when...`（逐字切）；
- 对：`[In an era] / [when digital distractions are prevalent]...`（按意群读整块）。
像看微信消息一样读整句，不拆字。

## 法律条文案例：英文就是「傻瓜游戏规则」

**原句**
> *Any person who, without lawful authority, intentionally destroys or damages any property belonging to another shall be guilty of an offense and liable on conviction to a fine not exceeding $10,000 or to imprisonment for a term not exceeding 2 years.*

**拆解成「游戏规则」**
1. **谁干了啥？** → 有人故意砸坏别人的东西；
2. **有没有授权？** → 没有合法授权（"without lawful authority"）；
3. **会怎样？** → 罚款 ≤ 1 万，或 坐牢 ≤ 2 年（"if–then" 结构）。

这就是英文典型的「如果—那么」逻辑：先定义触发条件（if），再给出后果（then）。法律英语把每个责任边界都写死在字面上，正是低语境语言「作者全责制」的极端体现。

## 终极真相：英文不是艺术，是工具

- 中文是诗 → 适合抒情、写人生感悟；
- 英文是说明书 → 适合说理、解决问题。

**学英文 = 给大脑装第二套系统**：需要感性时用中文（温情脉脉），需要理性时用英文（手术刀精准）。

**行动开关**：下次读英文，先问自己——「这是什么剧场？主角是谁？规则是什么？」所有答案都在纸上，别猜。别死记硬背，开始逆向工程。欢迎来到英文的底层世界——这里没有玄学，只有逻辑。
